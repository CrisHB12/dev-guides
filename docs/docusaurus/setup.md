# Setting up Docusaurus with Bun

Docusaurus is a modern static site generator. Using **Bun** makes the installation and build process significantly faster.

## Installation

To create a new project from scratch, run:

```bash
bunx create-docusaurus@latest my-website classic --typescript
```

## Essential Commands

- Install	-> bun install
- Development -> bun run start
- Build -> bun run build
- Serve -> bun run serve
